NBA 2k

Jesteś specem od uczenia maszynowego i sieci neuronowych, tyle że... Twój kot jest w tym od Ciebie lepszy.
Rozwiązuj łamigłówki, aby stworzyć translator języka czworonogów na mowę ludzi. Zbij majątek, kupuj kotu stroje i opanuj uczenie maszynowe!
to symulacyjna gra logiczna o rzeczach wymagających wysiłku szarych komórek: uczeniu maszynowym, sieciach neuronowych, danych big data i sztucznej inteligencji. Ale i tak ostatecznie liczy się to, czy zdołasz zrozumieć swojego kota. W tej grze wcielasz się w programistę, który przypadkowo odkrył, że jego kot dysponuje wyjątkowym talentem koderskim. Sęk w tym, że zwierzę nie potrafi zbyt dobrze mówić po ludzku. I tak oto koder (czyli Ty!) musi świetnie opanować uczenie maszynowe i za pomocą wizualnych narzędzi programistycznych stworzyć system rozpoznawania mowy przekładający język czworonogów na mowę ludzi.

Wymagania zalecane

  1. system - 64-bitowy Windows 7, 64-bitowy Windows 8 (8.1) lub 64-bitowy Windows 10
  2. procesor - Intel® Core™ i5-4430 z 3 GHz / AMD FX-8370 z 3,4 GHz lub lepszy
  3. pamięć RAM - 8 GB RAM
  4. karta graficzna - NVIDIA® GeForce® GTX 770 2GB / AMD® Radeon™ R9 270 2GB lub lepsza
  5. pamięć - 110 GB

Wymagania minimalne

  • system - 64-bitowy Windows 7, 64-bitowy Windows 8 (8.1) lub 64-bitowy Windows 10
  • procesor - Intel® Core™ i3-530 z 2,93 GHz / AMD FX-4100 z 3,60 GHz lub lepszy
  • pamięć RAM - 4 GB RAM
  • karta graficzna - NVIDIA® GeForce® GT 450 1GB / AMD® Radeon™ HD 7770 1GB lub lepsza
  • pamięć - 110 GB
